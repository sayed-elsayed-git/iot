Lab - Create a Simple Game with Python IDLE

Objectives

In this lab, you will create a simple game using Python IDLE.

Part 1: Create a Simple Game with Python IDLE

Part 2: Reflection

Required Resources

=   A modern personal computer with sufficient RAM with internet access.

=   Python / IDLE installed.

Instructions

The goal of the game is to find a number chosen by a player within 0 and 1024 using bisection method.

Part 1: Create a Simple Game with Python IDLE

a.     Start IDLE. (In Windows, click Start > Python3.x > IDLE, when x is the release version.

b.     Create a new file in IDLE. Click File -> New File to open a new (Untitled) Python script.

c.     Copy and paste the following script into the new file.

y=input('Enter an integer between 0 and 1024 -- ')

x=int(y)

a=0

b=1024

test=True

if x == 0:

    print('Your number is 0 , thank you for playing.')

    test=False

else:

    if x == 1024:

        print('Your number is 1024, thank you for playing.')

        test=False

 

    while test == True:

        n=int((a+b)/2)

        if n == x:

            print('Your number is ', n, ', thank you for playing.')

            break

        else:

            if n < x:

                a=n

            else:

                b=n

d.     Click File > Save, save the current script as game.py in your C:/Users/username/Documents directory. Click Save to continue.

e.     Click Run > Run Module (or press F5). The Shell window will display the result.

f.      Troubleshoot if an error occurs during code syntax evaluation.

2.     How would you catch a player who enters a float number?

Answer Area

The input should be verified to be an integer immediately after the player enters the number. The user input is a string. So you will have to verify that are only numerical values in the input string. For example, y.isnumeric() can test if the user input has only numerical digits.

Here is an example of the code:
